# Stellar Burgers
## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/Alexkucepalov/Stellar_burgers.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd Stellar_burgers
   ```

3. Установите зависимости:
   ```bash
   npm install
   ```

## Запуск

Для запуска приложения используйте команду:
```bash
npm run start
```
